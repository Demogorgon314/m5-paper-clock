USB 配置

插上 USB 后，在 Chrome 中授权串口，保存 Wi-Fi、时区和时钟样式。

  1. 连接设备
  2. 扫描并选择 Wi-Fi
  3. 保存并连接
  4. 同步时间并回到时钟页

需要 Chrome 或 Edge，并从 localhost 或 HTTPS 打开。

设置

保存后会写入设备存储，重启后继续生效。

Chrome 会记住你授权过的串口，点“恢复已授权设备”就能直接连回去。

设备重启后，串口可能会短暂断开，再点一次恢复即可。
